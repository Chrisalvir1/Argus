# Argus v1.4.6 — Simetría responsive y acciones SOS compactas

Esta versión de lanzamiento mejora la consistencia visual y la adaptabilidad responsive del panel de seguridad Argus Home Hub.

### 🌟 Cambios en esta versión

- **Simetría y Organización:** La cuadrícula principal del panel ha sido reorganizada en filas equilibradas.
- **Acciones SOS Compactas:** Se introdujo una cuadrícula adaptable con altura máxima y scroll interno independiente para las acciones SOS. Los nombres largos ahora se truncan con seguridad y muestran el título completo al posicionar el cursor sobre ellos.
- **Optimización Móvil y Textos Largos:** Adaptaciones para que la vista móvil se reordene a una sola columna sin desbordamientos y se prevengan desbordes de textos largos en diferentes idiomas.
