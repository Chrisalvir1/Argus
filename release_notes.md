# Argus v1.5.2 — Local-First Operations & Encrypted Recovery

Esta actualización introduce el motor WebGL nativo para visualización del clima, el asistente inteligente Copilot local en 7 idiomas, mayor resiliencia ante cortes eléctricos y copias de seguridad de configuración cifradas localmente.

### 🌟 Cambios en esta versión

- **WebGL Nativo en GPU:** Renderizado de partículas de lluvia, nieve, niebla y relámpagos sin dependencias y con fallback seguro a CSS.
- **Resiliencia ante Apagones:** Persistencia local que recupera de inmediato el estado de alarma y horarios programados.
- **Centro de Salud y Energía:** Monitoreo exhaustivo de sensores con baterías y telemetría de red eléctrica estrictamente reales.
- **Línea de Tiempo Forense:** Registro detallado de eventos indicando actores, severidad y marcas de tiempo exactas.
- **Copilot Multilingüe Local:** Consultas de soporte y control directamente procesadas de manera segura en tu servidor local.
- **Copias Cifradas con AES-256-GCM:** Backups de configuración robustos para protección de datos sensibles.
- **API de Extensiones y Accesibilidad:** Compatibilidad mejorada para lectores de pantalla y WebSocket API para desarrolladores ([docs/EXTENSIONS_API.md](file:///Users/chrisalvir/Desktop/GITHUB%20PROJECT/argus/docs/EXTENSIONS_API.md)).
