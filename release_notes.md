# Argus v1.4.8 — Rediseño compacto de Control de Acceso y Usuarios

Esta actualización incluye un rediseño compacto para la sección de Control de Acceso y Usuarios, mejorando la visualización de la información.

### 🌟 Cambios en esta versión

- **Rediseño compacto de “Control de Acceso y Usuarios”:** El panel ahora muestra un resumen del PIN maestro y usuarios configurados.
- **Formularios bajo demanda:** Los formularios de usuarios y PIN se abren solo bajo demanda.
- **Funcionalidad conservada:** Se conserva toda la funcionalidad existente sin cambios estructurales.
