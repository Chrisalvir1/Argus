## 🗨️ Home Assistant Community Forum Post

**Title:** Argus Home Hub v1.3.3 — Premium alarm panel with Liquid Glass UI, 7 languages, guest PINs with expiry, animated weather, and more

**Body:**

Hi HA community! 👋

I'm excited to share the latest release of **[Argus Home Hub](https://github.com/Chrisalvir1/Argus)** (v1.3.3), a full custom integration I've been building as a premium alternative to Alarmo.

### What's new in v1.3.3
- **Liquid Glass Lovelace card** with built-in PIN keypad (auto-shows when Master PIN is active)
- **Instant language switching** across 7 languages (no reload)
- **Admin-only PIN reset** with audit trail entry
- **Physical file storage** for backgrounds (migrates away from Base64)

### Why Argus instead of Alarmo?

Alarmo is great but I wanted more. Argus adds:

- 🎨 **Premium Liquid Glass UI** (glassmorphism, backdrop-blur, smooth animations)
- 🌤️ **Animated weather backgrounds** — pure CSS, zero processing overhead
- 🔑 **Guest codes with expiration dates** — perfect for temporary access
- 🚨 **Panic/SOS mode** — slide-to-confirm immediate alarm
- 📋 **Rich audit log** — who armed/disarmed + which sensor triggered + timestamp
- 🔋 **Battery monitoring HUD** — auto-scans all intrusion sensors, alerts below 20%
- 🌍 **7 languages** with instant in-app switching
- 📱 **Fullscreen + multi-instance** for wall-mounted tablets
- 🏠 **HomeKit bridge status** built in

### Installation (HACS)

1. HACS → Integrations → `⋮` → **Custom repositories**
2. Add `https://github.com/Chrisalvir1/Argus` → Category: **Integration**
3. Search **Argus** → Download → Restart HA
4. **Settings → Devices & Services → Add Integration → Argus**
5. Follow the setup wizard — no YAML needed!

**Lovelace card:**
```yaml
type: custom:argus-card
entity: alarm_control_panel.argus
```

**Full admin panel (for tablets):**
```yaml
type: custom:argus-panel
```

Happy to answer any questions! And if you find a bug or have a feature request, [open an issue on GitHub](https://github.com/Chrisalvir1/Argus/issues). ⭐
